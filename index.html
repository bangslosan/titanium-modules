Extensive List of Titanium Modules!

This list (should) contain all proper Titanium modules, preferable well documented but at least working and up-to-date. If you find a module doesn't work (anymore) or the list is missing modules, feel free to raise an issue or (even better) submit a PR!

This list is maintained thanks to my backers at Patreon. Fund me there!

Push Notifications

  • Ti.CloudPush - Module by appcelerator to receive pushes
  • Ti.Goosh - Receiving push notifications on Android

Image handling

  • Ti.ImageFactory - Resize, crop and more image manipulation
  • av.imageview - Using content mode for displaying images
  • com.mfogg.squarecamera - Great camera module with Barcode support

Barcode scanners

  • ti.barcode - Barcode scanner by Appcelerator
  • Acktie-Mobile-Android-Barcode - Fast Android barcode scanner
  • com.mfogg.squarecamera - Great camera module with Barcode support

Caching

  • To.ImageCache - Store images locally and define expire time per image

Analytics

  • Ti.GA - Google Analytics
  • ti.flurry - Flurry (crash)analytics module by Appcelerator

HTTP clients

  • XHR.js - Great and simple way to execute HTTP calls
  • RESTe - Another great way to execute HTTP calls and populate collections on the fly

Other

  • de.manumaticx.circularprogress - Circular progress bars
  • de.marcelpociot.autofocus - No auto focus on textfields on Android anymore
  • bencoding.alarmmanager - Android Alarm manager

Widgets

  • To.BounceView - A Titanium View that can bounce using animations.
  • nl.fokkezb.infiniteScroll - Paging in ListViews/TableViews
  • nl.fokkezb.pullToRefresh - Pull to refresh for ListViews/TableViews
  • ti.searchbar - Searchbar
